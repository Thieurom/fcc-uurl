URL Shortener

Feature

Passing a URL as a parameter, then receiving a shortened URL in the JSON response.

If passing an invalid URL , the JSON response will contain an error instead.

When visiting that shortened URL, it will redirect to the original link.

Example usage:

  • https://u-url.herokuapp.com/shorten/https://www.freecodecamp.com/
  • https://u-url.herokuapp.com/shorten/http://foo.com

Example response:

{ original_url: http://foo.com/, “short_url”: “https://u-url.herokuapp.com/12345” }
